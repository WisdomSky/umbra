Umbra JS

Umbra JS - Create semantic html presentation with custom elements and templates

View on GitHub Download .zip Download .tar.gz

Umbra JS is a very simple, straight-forward and easy to use jQuery plug-in for creating a modularized html presentation. HTML Custom Elements is a capability for creating your own custom HTML tags and elements. Browsers with Custom Elements support recognize any invalid html tag you write in your html file, which means you retain the ability to control that invalid element, however on non-supporting browsers, elements with invalid tag will be just completely ignored.

Umbra JS combines custom elements with templating system. With it, you can attach a template to each custom elements you defined in your html.

How to use?

Duhh?! It's never been so easy!

Let's say you have a div with a span inside and want to represent it as a single custom element named "my-element"

Your HTML should look like:

<my-element>John</my-element>

And your javascript should look like:

<script>
         $(document).umbra({
              tag: {
                  "my-element" : "<div>My name is: <span>{{content}}</div>"
              }
         });
</script>

Umbra JS also supports external file templates.

<script>
         $(document).umbra({
              tag: {
                  "my-element" : "external-template.html"
              }
         });
</script>

And also supports closures for much more advanced manipulation. The context of the closure is the current custom element being processed. Take note that the closure should return either html text or jquery object!

<script>
         $(document).umbra({
              tag: {
                  "my-element" : function() {
                        var $sample = $("<div>");

                        $sample.html( "My name is: "+ $(this).html() );

                        return $sample;
              }
         });
</script>

Now, let's combine html and javascript part:

<html>
    <head>
        <title>Hey Umbra JS!</title>
        <script src="jquery.min.js"></script>
        <script src="umbra.jquery.js"></script>
        <script>
                $(document).umbra({
                  tag: {
                      "my-element" : "<div>My name is: <span>{{content}}</div>"
                  }
                });
        </script>

    </head>
    <body
        <my-element>John</my-element>
    </body>
</html>

I have a bad news though!

Umbra JS is natively supported only by Google Chrome. :(

But wait! there's a workaround! I don't want you to feel bad for not having Umbra JS in different browsers. So I'll tell you how to make it work on other browsers.

You need to download Web Components Polyfills library and include the webcomponentsjs.js just above the umbra.jquery.js like this:

<html>
    <head>
        <title>Hey Umbra JS!</title>
        <script src="jquery.min.js"></script>
        <script src="webcomponentsjs/webcomponents.js"></script>
        <script src="umbra.jquery.js"></script>
        <script>
                $(document).umbra({
                  tag: {
                      "my-element" : "<div>My name is: <span>{{content}}</div>"
                  }
                });
        </script>

    </head>
    <body
        <my-element>John</my-element>
    </body>
</html>

And voila! Umbra JS is now working on other browsers! :D
